# Bootstrap 5 - Справочник стилей проекта

Этот документ содержит все Bootstrap классы, используемые в проекте Quiz, с описанием и примерами.

## Layout (Разметка)

### Container
- **`container`** - Адаптивный контейнер с фиксированной максимальной шириной
  - Автоматически центрируется
  - Имеет отступы по бокам
  - Пример: `<div class="container">...</div>`

### Grid System (Сетка)
- **`row`** - Создает строку для колонок
  - Используется вместе с `col-*`
  - Пример: `<div class="row">...</div>`

- **`col-sm-6`** - Колонка шириной 50% на экранах ≥576px
  - sm = small (маленькие экраны)
  - 6 из 12 колонок = 50%

- **`col-lg-4`** - Колонка шириной 33.33% на экранах ≥992px
  - lg = large (большие экраны)
  - 4 из 12 колонок = 33.33%

## Flexbox (Гибкая разметка)

### Display
- **`d-flex`** - Включает flexbox для элемента
  - Дочерние элементы выстраиваются в ряд
  - Базовый класс для flex-контейнера

### Direction (Направление)
- **`flex-column`** - Вертикальное расположение элементов
  - По умолчанию flex использует горизонтальное направление
  - Этот класс меняет на вертикальное

- **`flex-wrap`** - Разрешает перенос элементов на новую строку
  - Если элементы не помещаются, они переносятся

### Justify Content (Выравнивание по главной оси)
- **`justify-content-between`** - Распределяет элементы с максимальным расстоянием между ними
  - Первый элемент в начале, последний в конце
  - Пример: кнопки по краям хедера

- **`justify-content-center`** - Центрирует элементы по горизонтали

### Align Items (Выравнивание по поперечной оси)
- **`align-items-center`** - Центрирует элементы по вертикали
  - Часто используется вместе с `d-flex`

- **`align-items-start`** - Выравнивает элементы по верхнему краю

### Align Middle
- **`align-middle`** - Вертикальное выравнивание по центру (для таблиц)

### Gap (Отступы между элементами)
- **`gap-1`** - Отступ 0.25rem (4px) между flex-элементами
- **`gap-2`** - Отступ 0.5rem (8px) между flex-элементами
- **`gap-3`** - Отступ 1rem (16px) между flex-элементами

### Flex Grow
- **`flex-grow-1`** - Элемент растягивается, занимая доступное пространство

## Spacing (Отступы)

### Padding (Внутренние отступы)
- **`p-4`** - Padding со всех сторон 1.5rem (24px)
- **`p-5`** - Padding со всех сторон 3rem (48px)
- **`py-2`** - Padding сверху и снизу 0.5rem (8px)
  - y = vertical (вертикальный)
- **`py-4`** - Padding сверху и снизу 1.5rem (24px)
- **`px-3`** - Padding слева и справа 1rem (16px)
  - x = horizontal (горизонтальный)

### Margin (Внешние отступы)
- **`m-2`** - Margin со всех сторон 0.5rem (8px)
- **`mb-0`** - Margin снизу 0 (убирает отступ)
  - b = bottom (снизу)
- **`mb-2`** - Margin снизу 0.5rem (8px)
- **`mb-3`** - Margin снизу 1rem (16px)
- **`mb-4`** - Margin снизу 1.5rem (24px)
- **`mt-2`** - Margin сверху 0.5rem (8px)
  - t = top (сверху)
- **`mt-3`** - Margin сверху 1rem (16px)
- **`ms-auto`** - Margin слева auto (прижимает элемент вправо)
  - s = start (начало, слева в LTR)

## Typography (Типографика)

### Headings (Заголовки)
- **`h3`** - Стиль заголовка h3 (можно применить к любому элементу)
- **`h4`** - Стиль заголовка h4

### Text Alignment (Выравнивание текста)
- **`text-center`** - Центрирует текст

### Text Colors (Цвета текста)
- **`text-muted`** - Приглушенный серый цвет (#6c757d)
- **`text-primary`** - Основной цвет (синий #0d6efd)
- **`text-success`** - Цвет успеха (зеленый #198754)
- **`text-danger`** - Цвет ошибки (красный #dc3545)
- **`text-secondary`** - Вторичный цвет (серый #6c757d)
- **`text-body-tertiary`** - Третичный цвет текста (светло-серый)

### Text Transform (Преобразование текста)
- **`text-uppercase`** - Преобразует текст в ЗАГЛАВНЫЕ БУКВЫ

### Font Size (Размер шрифта)
- **`small`** - Уменьшенный размер шрифта (0.875em)
- **`fs-5`** - Font size 5 (1.25rem)

### Font Weight (Толщина шрифта)
- **`fw-bold`** - Жирный шрифт (700)
- **`fw-semibold`** - Полужирный шрифт (600)

### Font Style (Стиль шрифта)
- **`fst-italic`** - Курсивный текст

## Buttons (Кнопки)

### Base
- **`btn`** - Базовый класс для всех кнопок (обязательный)

### Button Variants (Варианты кнопок)
- **`btn-primary`** - Основная кнопка (синяя)
- **`btn-success`** - Кнопка успеха (зеленая)
- **`btn-danger`** - Кнопка опасности (красная)
- **`btn-secondary`** - Вторичная кнопка (серая)
- **`btn-outline-primary`** - Кнопка с обводкой (синяя рамка, прозрачный фон)
- **`btn-outline-danger`** - Кнопка с обводкой (красная рамка)

### Button Sizes (Размеры кнопок)
- **`btn-sm`** - Маленькая кнопка

### Button Utilities
- **`btn-close`** - Кнопка закрытия (крестик ×)
- **`w-100`** - Ширина 100% (кнопка на всю ширину контейнера)

## Cards (Карточки)

- **`card`** - Контейнер карточки с рамкой и скругленными углами
- **`card-body`** - Основное содержимое карточки с padding
- **`card-title`** - Заголовок карточки
- **`card-text`** - Текст карточки
- **`h-100`** - Высота 100% (карточка растягивается на всю высоту)

## Shadows (Тени)

- **`shadow`** - Стандартная тень
- **`shadow-sm`** - Маленькая тень (более тонкая)

## Alerts (Уведомления)

- **`alert`** - Базовый класс для уведомлений
- **`alert-success`** - Зеленое уведомление (успех)
- **`alert-danger`** - Красное уведомление (ошибка)

## Badges (Значки)

- **`badge`** - Базовый класс для значков
- **`bg-primary-subtle`** - Светло-синий фон
- **`bg-success-subtle`** - Светло-зеленый фон
- **`bg-danger-subtle`** - Светло-красный фон
- **`rounded-pill`** - Скругленная форма (таблетка)

## Forms (Формы)

### Form Controls
- **`form-label`** - Стиль для label элементов
- **`form-control`** - Стиль для input и textarea
- **`form-control-sm`** - Маленький input

### Form Check (Чекбоксы и радио)
- **`form-check`** - Контейнер для checkbox/radio
- **`form-check-input`** - Стиль для checkbox/radio input
- **`form-check-label`** - Label для checkbox/radio

## Tables (Таблицы)

- **`table`** - Базовый класс для таблиц
- **`table-responsive`** - Делает таблицу прокручиваемой на маленьких экранах
- **`table-hover`** - Подсвечивает строку при наведении
- **`table-light`** - Светлый фон для thead

## Borders (Рамки)

- **`border`** - Добавляет рамку со всех сторон
- **`border-start`** - Рамка слева (start = left в LTR)
- **`border-4`** - Толщина рамки 4px
- **`border-primary`** - Синяя рамка
- **`border-success`** - Зеленая рамка
- **`border-danger`** - Красная рамка
- **`rounded`** - Скругленные углы

## Background (Фон)

- **`bg-light`** - Светло-серый фон (#f8f9fa)
- **`bg-white`** - Белый фон

## Display (Отображение)

- **`d-block`** - Display: block (элемент занимает всю ширину)

## Sizing (Размеры)

- **`min-vh-100`** - Минимальная высота 100vh (100% высоты viewport)
  - Используется для полноэкранных страниц (например, login)

## Position (Позиционирование)

- **`position-absolute`** - Абсолютное позиционирование
- **`top-0`** - Позиция сверху 0
- **`end-0`** - Позиция справа 0 (end = right в LTR)

## Комбинации классов (примеры из проекта)

### Центрированная форма логина
```html
<body class="bg-light d-flex justify-content-center align-items-center min-vh-100">
```
- Светлый фон на всю страницу
- Flexbox для центрирования
- Центрирование по горизонтали и вертикали
- Минимальная высота = высота экрана

### Хедер с кнопками
```html
<div class="d-flex justify-content-between align-items-center mb-4">
```
- Flexbox контейнер
- Элементы по краям (заголовок слева, кнопки справа)
- Вертикальное выравнивание по центру
- Отступ снизу

### Карточка квиза
```html
<div class="card h-100 shadow-sm">
    <div class="card-body d-flex flex-column">
```
- Карточка на всю высоту родителя
- Легкая тень
- Содержимое в виде вертикального flex-контейнера

### Адаптивная сетка
```html
<div class="row g-3">
    <div class="col-sm-6 col-lg-4">
```
- Строка с отступами между колонками (gap-3)
- На маленьких экранах: 2 колонки (50% каждая)
- На больших экранах: 3 колонки (33.33% каждая)

### Значок с количеством вопросов
```html
<span class="badge bg-primary-subtle text-primary rounded-pill">5 questions</span>
```
- Значок в форме таблетки
- Светло-синий фон с синим текстом
